Computational Physics

Being able to write basic computer code is becoming more important in physics and astrophysics research, as well as in industry, yet many students are not able to do so. This class will give students an introduction to computer science, with a focus on physics use cases. No prior knowledge of computer programming is required.

A majority of the code written for this class will be in Python, as that is the new and cool language these days, and it is much easier to understand than older languages. However, all the skills and techniques learned through this class are directly transferrable to writing code in any programming language.

This class is broken into 3 parts:

  • Introduction (Lessons 1-3; Chapters 1-5): This section will bring you up to speed on how to write programs in Python, and how to do basic calculus with computers.
  • Advanced Topics (Lessons 4-8; Chapters 6-10): This section will focus on many real-world applications of physics programming: solving sets of equations, Fourier transforms, and differential equations.
  • Added Content (Lessons 9+): This section is provided to enhance your skills in many currently-relevant areas, such as parallel computing and database systems.

This course is written to be a companion to Computational Physics by Mark Newman, an excellent book on this topic. All textbook chapters included in the Introduction part of the course are available for free on the author's website.
